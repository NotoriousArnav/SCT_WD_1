# Task 01 – Responsive Landing Page with Fixed Interactive Navigation

**SkillCraft Technology Assessment**

A clean, modern, fully responsive landing page built with pure HTML, CSS, and vanilla JavaScript — **single file**, **no external dependencies**.

## Features

- Fixed sticky navigation (always visible)  
- Transparent → solid glass-morphism background on scroll  
- Smooth hover effects with animated underline  
- Fully responsive (mobile + desktop)  
- Mobile hamburger menu  
- Modern aesthetics with gradients and blur effects  

## How to Run

```bash
npx vite
```

Then Head to http://localhost:5173
